Система учета финансов
======================
Система предназначена для контроля доходов и расходов пользователя.

Возможности системы:
--------------------
* Создание и удаление счетов
* Создание и удаление доходов и расходов
* Просмотр журнала всех операций
* Просмотр всех доступных средств на счетах
* Просмотр актуального курса валют (с использованием стороннего API)
* Работоспособность при отсутствии подключения к интернету (кроме отображения курса валют)
* Система использует LocalStorage в Google Chrome

Технологии, используемые в системе:
-----------------------
* Javascript
* Jquery
* Node.js
* Express Js
* Twig
* Materialize

Запуск под Ubuntu:
------------------
###### 1. Cклонировать проект из репозитория ######

###### 2. Выполнить в командной строке: ######
```sudo apt-get update && sudo apt-get install -y nodejs nodejs-legacy npm```

###### 3. Выполнить в командной строке (из директории проекта): ######
```npm install && node index.js```
###### 4. Запустить браузер и в адресной строке перейти по: ######
```http://localhost:3000/```
